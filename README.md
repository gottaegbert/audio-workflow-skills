# VocalFlow Studio

[中文说明](README.zh-CN.md) | [English](README.en.md)

VocalFlow Studio turns media links, local audio/video files, and vocal stems into karaoke practice assets, separated vocals/instrumentals, and subtitle files.

This repository contains:

- A desktop app prototype in [apps/desktop](apps/desktop).
- CLI helpers for subtitles, lyrics, MP3 download, and vocal separation.
- Codex skills for repeatable audio workflows.
